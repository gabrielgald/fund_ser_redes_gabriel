# Website Moderno - Gabriel

Este é um site simples e estilizado criado para demonstração.

## Estrutura do Projeto

- `index.html`: O arquivo principal contendo a estrutura HTML do site.
- `styles.css`: O arquivo de estilos para personalizar a aparência do site.

## Imagens

As imagens utilizadas no site são provenientes do [Unsplash](https://unsplash.com/) e são carregadas dinamicamente a partir de URLs públicas:
- Paisagem: `https://source.unsplash.com/300x200/?nature,landscape`
- Arquitetura: `https://source.unsplash.com/300x200/?architecture`
- Tecnologia: `https://source.unsplash.com/300x200/?technology`
- Arte: `https://source.unsplash.com/300x200/?art`
- Cidade: `https://source.unsplash.com/300x200/?city`

Se você desejar substituir essas imagens, substitua as URLs no arquivo `index.html` pela localização das suas imagens.

## Como Usar

1. Extraia o conteúdo do arquivo ZIP.
2. Abra o arquivo `index.html` em um navegador para visualizar o site.

Se precisar de suporte adicional, entre em contato.
